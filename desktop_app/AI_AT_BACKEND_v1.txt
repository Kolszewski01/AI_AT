### 5. NASTĘPNE KROKI - DESKTOP APP NA WINDOWS

**Desktop app wymaga:**
- Python 3.10+ (na Windows)
- PyQt6
- Połączenie z backendem (który działa na WSL)

**Plan uruchomienia na Windows:**
1. Sklonować repo: git clone https://github.com/Kolszewski01/AI_AT_DESKTOP.git
2. Utworzyć venv: python -m venv venv
3. Aktywować: venv\Scripts\activate
4. Zainstalować: pip install -r requirements.txt
5. Skonfigurować adres backendu (WSL IP: localhost lub 127.0.0.1:8000)
6. Uruchomić: python main.py

**Adres backendu z Windows:**
- Backend w WSL jest dostępny na: http://localhost:8000
- WSL2 automatycznie forwarduje porty do Windows
